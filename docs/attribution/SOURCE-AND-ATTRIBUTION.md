# Fonte e atribuição das bandeiras

## Fonte

Os arquivos SVG de bandeiras deste pacote foram informados pelo usuário como provenientes de:

- Flagpedia.net
- Página de download: `https://flagpedia.net/download`
- Página de termos: `https://flagpedia.net/terms`

## Uso informado pela fonte

O Flagpedia informa que suas imagens de bandeiras são de domínio público e podem ser usadas livremente. Nas páginas de download, o site também informa que o pacote inclui 254 bandeiras, é gratuito para uso comercial e não comercial, é baseado em arquivos vetoriais do Wikimedia Commons e que um backlink para `https://flagpedia.net` é apreciado.

## Atribuição sugerida para o Flag Atlas

Use esta frase no README, na página “Sobre” ou em uma seção de créditos do app:

> Flag images sourced from Flagpedia.net. Flagpedia states that its flag images are in the public domain. Backlink appreciated: https://flagpedia.net

Versão em português:

> Imagens de bandeiras obtidas do Flagpedia.net. O Flagpedia informa que suas imagens de bandeiras estão em domínio público. Backlink apreciado: https://flagpedia.net

## Observação sobre o projeto

Esta documentação cobre apenas a origem das bandeiras. A licença do código, design, textos e demais assets do Flag Atlas deve ser definida separadamente no repositório principal do projeto.
