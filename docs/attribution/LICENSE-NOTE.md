# Nota de licença e fonte

Fonte informada pelo usuário para os SVGs originais: Flagpedia.net.

Segundo o Flagpedia.net, as imagens de bandeiras disponibilizadas no site estão em domínio público e podem ser usadas livremente, inclusive para uso comercial e não comercial. O site também informa que as bandeiras são baseadas em arquivos vetoriais do Wikimedia Commons e que aprecia um backlink para `https://flagpedia.net`.

Este pacote corrigido apenas reorganiza, valida e estrutura os arquivos recebidos para uso no projeto Flag Atlas.

## Recomendação de atribuição

Mesmo não sendo apresentada como obrigatória pelo Flagpedia, recomenda-se manter uma atribuição simples no projeto:

> Flag images sourced from Flagpedia.net. Flagpedia states that its flag images are in the public domain. Backlink appreciated: https://flagpedia.net

## Observação

Esta nota não é uma licença do código do Flag Atlas. Ela documenta apenas a origem declarada dos arquivos de bandeiras recebidos e a informação pública exibida pelo Flagpedia.net no momento da organização deste pacote.
