Flag Atlas Design System · duas direções de identidade

Modern cartographic learning game

Verde-topográfico + ocre, Nunito refinada, ícones de linha e a bandeira como protagonista. Escolha uma direção (1a ou 1b) — depois eu expando a escolhida no spec completo de tokens para React + Tailwind.

Opção 1a

Meridian

Pergaminho & Pinho · quente, editorial, tátil

Base creme de mapa antigo, verde-pinho profundo e ocre. Curvas de nível sutis, selos de carimbo e sombras macias. Divertido, mas com peso de atlas de verdade.
