# ADR 0001 — Vite + React + TypeScript + PWA

## Status

Accepted

## Contexto

Flag Atlas é uma aplicação web interativa, responsiva e offline-first. O projeto precisa crescer com qualidade sem começar pesado demais.

## Decisão

Usar Vite, React e TypeScript como base da aplicação.

Configurar PWA desde o MVP.

## Consequências

- Build rápido.
- Boa experiência de desenvolvimento.
- Ecossistema sólido.
- TypeScript ajuda a proteger lógica de dados e sessão.
- PWA exige testes e atenção a cache/service worker.
