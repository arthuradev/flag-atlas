# /handoff

Use este comando para entregar um resumo de estado ao usuário.

## Incluir

- Fases concluídas.
- Commits feitos.
- Como rodar localmente.
- Como rodar testes.
- Como fazer deploy.
- O que está implementado.
- O que ficou fora do MVP.
- Pendências e riscos.

## Não incluir

- Promessas vagas.
- Funcionalidades não implementadas como se estivessem prontas.
- Afirmações sem validação.
