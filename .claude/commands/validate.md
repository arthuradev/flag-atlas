# /validate

Use este comando para validar o estado atual do projeto.

## Rodar

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

Se houver app navegável:

```bash
pnpm e2e
```

## Conferir manualmente

- App abre.
- Onboarding funciona.
- Home funciona.
- Treino funciona.
- Coleção funciona.
- Configurações funcionam.
- Offline funciona após primeiro carregamento.

## Saída esperada

- Resultado dos comandos.
- Problemas encontrados.
- Correções realizadas.
- Pendências restantes.
